# ✨ 星空游乐园 ✨

> *"在浩瀚星空下，用代码编织浪漫与欢乐。"*

一个充满浪漫气息的互动网页合集，包含 **星空告白** 页面和两款经典小游戏 —— **幸运猜数字** 与 **贪吃蛇**。所有页面共享统一的深邃星空主题，通过一个优雅的导航首页串联，带来沉浸式的夜空体验。

---

## 🌌 页面总览

| 页面 | 文件 | 说明 |
|:---:|:---:|:---|
| 🏠 导航首页 | `index.html` | 项目主入口，星空背景 + 毛玻璃导航按钮，一键直达各子页面 |
| 💫 星空下的告白 | `love.html` | 动态星空背景、闪烁星星、飘落爱心、呼吸发光文字 |
| 🎲 幸运猜数字 | `guess-number.html` | 经典猜数字游戏，带历史记录与智能提示 |
| 🐍 经典贪吃蛇 | `snake-game.html` | 星空主题贪吃蛇，支持加速与暂停 |

---

## 🎮 功能亮点

### 💫 星空下的告白

- 🌠 **动态星空背景** — 150+ 颗随机生成的星星，闪烁频率各不相同
- 💖 **飘落爱心雨** — 持续生成、随机大小与飘动轨迹的爱心动画
- ✨ **呼吸发光文字** — 告白文字以心跳般的节奏脉动，梦幻感十足

### 🎲 幸运猜数字

- 🏆 **历史最高分记录** — 使用 `localStorage` 持久化保存最佳猜测次数
- 📐 **智能范围提示** — 每次猜测后实时更新当前数字范围，辅助决策
- 🎯 **即时反馈** — "太大了" / "太小了" / "恭喜你" 三色动态反馈

### 🐍 经典贪吃蛇

- ⚡ **动态加速机制** — 每吃 3 个食物自动加速，越玩越刺激
- ⏱ **3 秒开局 / 恢复倒计时** — 开始游戏与暂停恢复均有炫酷数字倒计时
- ⏸ **空格键暂停** — 随时暂停/恢复，从容应对每一个转弯
- 🥇 **最高分记录** — `localStorage` 持久化保存历史最佳成绩
- 📱 **触屏方向键** — 移动端自动显示虚拟方向键，完美适配手机操作

## 🚀 如何运行

这是一个 **纯前端项目**，无需安装任何环境、无需构建工具、零依赖。

直接用浏览器打开 `index.html` 即可体验全部功能 👇

```bash
# 方式一：直接双击打开
# 双击 index.html 即可在默认浏览器中运行

# 方式二：使用 Python 启动本地服务器（可选）
python3 -m http.server 8000
# 然后打开 http://localhost:8000

# 方式三：使用 Node.js 启动本地服务器（可选）
npx serve .
# 然后打开终端中显示的地址
```

## 🛠 技术栈

| 技术 | 用途 |
|:---:|:---|
| **HTML5** | 页面结构与 Canvas 画布 |
| **CSS3** | 渐变背景、毛玻璃效果、CSS 动画（星星闪烁 / 爱心漂浮 / 文字呼吸） |
| **JavaScript** | 原生 DOM 操作、游戏逻辑、`localStorage` 数据持久化 |

> 💡 纯原生实现，不依赖任何第三方库或框架，轻量且优雅。
